# 高涵个人求职主页

一个独立的纯前端 resume 项目，用于展示 AI 产品经理与通用产品经理方向的个人主页、项目案例和简历下载。

## 技术栈

- Vite
- React
- TypeScript
- Tailwind CSS
- lucide-react

## 本地开发

```bash
npm install
npm run dev
```

默认本地地址：

```text
http://127.0.0.1:5173/
```

## 构建检查

```bash
npm run lint
npm run build
```

## 静态资源

PDF 简历位于：

```text
public/高涵-杭州-产品经理.pdf
```

后续可将本项目部署到 GitHub Pages、Vercel 或 Netlify。
